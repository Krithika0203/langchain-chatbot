# 🤖 NeuraChat – AI Chatbot

✨ An intelligent chatbot built using **LangChain + LLaMA 3.3 + Streamlit**, powered by Groq for fast responses.

---

## 🚀 Features

* 💬 Interactive AI Chat Interface
* ⚡ Fast responses using Groq API
* 🎛️ Adjustable parameters (temperature, tokens)
* 🧠 Context-aware conversations
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* Python
* LangChain
* LLaMA 3.3 (via Groq)
* Streamlit
